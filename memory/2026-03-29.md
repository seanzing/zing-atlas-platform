- PrismaClient constructor takes `{ adapter }` — not `datasourceUrl`
- Single org_id `00000000-0000-0000-0000-000000000001` for Phase 1 single-tenant
- Seed script uses `npx tsx` (not ts-node) for reliable ESM/CJS interop
